# 🌸 Japanese Learning Station
### Minna no Nihongo — N5 Beginner Course

🔗 **Live Site → [https://hossainz26.github.io/Minna-no-Nihongo-N5/](https://hossainz26.github.io/Minna-no-Nihongo-N5/)**

---

A beginner-friendly Japanese learning website covering Hiragana, Katakana, N5 Vocabulary, Grammar, Listening, and Assignments. Pure HTML, CSS, and JavaScript — no frameworks, no build tools.

---

## 📄 Pages

| Page | Description |
|------|-------------|
| 🏠 Home | Welcome page with stats and feature overview |
| 🔤 Letters | Full Hiragana & Katakana charts + interactive quiz |
| 📚 Vocabulary | 150+ N5 words by category with flip cards |
| 📖 Grammar | 12 N5 grammar patterns with examples |
| 🎧 Listening | Upload and play audio files in the browser |
| 📝 Assignment | Upload PDFs/images + N5 study checklist |

## 🚀 Run Locally

Just open `index.html` in your browser — no server needed.

Or use VS Code → install **Live Server** extension → right-click `index.html` → **Open with Live Server**.
